Qui trovi tutti i piani mensili. Per aggiungere un nuovo piano, aggiungi un oggetto nell'array PIANI in fondo allo script, e aggiungi le settimane nell'array WEEKS con la chiave piano corrispondente. I dettagli degli allenamenti vanno nell'oggetto DETAILS.

Come aggiornare il piano

  1. Apri il file HTML in un editor di testo (es. VS Code, Notepad++).
  2. Trova l'array PIANI nello script. Aggiungi un nuovo oggetto con id, label, meta, gara.
  3. Trova l'array WEEKS. Aggiungi le nuove settimane con piano: 'id_del_tuo_piano'.
  4. Aggiungi i dettagli di ogni allenamento nell'oggetto DETAILS con chiavi univoche.
  5. Per marcare un allenamento come completato, aggiungi done: true al badge nella settimana.
  6. Salva il file e riaprilo nel browser. Nessun server necessario.
